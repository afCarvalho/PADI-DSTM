Para correr os servidores para clicar duas vezes em run_dstm.bat

Se por alguma razão isso não for possível, abra o mesmo ficheiro e siga cada uma das directorias
para encontrar os executáveis.

Os executáveis devem ser corridos pela mesma ordem que se encontram no ficheiro, ou seja, Master -> Server/s -> Client/s,
sendo no entanto possível executar novos servidores com o sistema já em execução.

No caso do PadIntServer necessitar de um porto específico basta passar o valor do porto como argumento da main.
Ex: .\PadIntServer.exe 2001
